Arquivo HTML

Basicamente o HTML é usado em praticamente todos os sites existentes. Qualquer página de um site que você for visitar será feita de HTML. Uma boa comparação seria com um esqueleto humano, o HTML é todo o esqueleto de um website, é praticamente impossível fazer um website sem usar HTML. Dito isto, para aqueles que querem se aventurar no mundo de Web Development ou pretende aprender HTML para algum outro uso, talvez Hobby, deverá ter o HTML em sua caixa de ferramenta pois ela é essencial para que se possa criar um Website.

Como criar um arquivo HTML ?

Para criar um arquivo HTML, você pode usar um editor de Texto tradicional como o Bloco de Notas que já vem no próprio windows ou simplesmente baixar editores de código na Internet, irei citar alguns no final deste Post. Para começar você pode dar qualquer nome para seu arquivo HTML, porém, como é página inicial você deve pôr o nome de index e salvá-lo com a extensão .html, assim o servidor irá identificar que aquela página index é a inicial. Basicamente o que você irá pôr primeiro no arquivo .html é o corpo do HTML. Veja na imagem os passos citados em cima:

<html>

<head>

<title>//Título</title>

</head>

<body>

//Conteúdo

</body>

</html>
  • HTML é a ordem de abertura do HTML;
  • Head é onde colocamos tudo que não aparecerá na página para o usuário ver;
  • Title é onde coloca o titulo da página, que aparecera na aba guia do navegador;
  • Body é onde todo o conteúdo do site será mostrado para o usuário ver.

Como abrir um arquivo HTML ?

Você deve sempre abrir o arquivo HTML com um navegador, pois o navegador é que lê e interpreta o arquivo HTML.
