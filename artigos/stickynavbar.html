Menu Fixado com Javascript

Primeiramente precisamos criar nosso menu de navegação. Aqui neste exemplo eu já criei meu menu de navegação para usá-lo como exemplo para poder fazer o sticky nav e também como podem ver na imagem, eu adicionei textos aleatórios para assim poder dar scroll na página e ver o resultado do menu fixado no topo da tela.

Criaremos agora no CSS uma classe chamada "sticky" e colocamos sua posição como fixed.

Agora, iremos fazer todo o trabalho com o javascript. Primeiro de tudo temos que selecionar de nosso DOM o nosso elemento de barra de menu lá no javascript, no meu caso a "id" do elemento é navbar.

Agora precisamos pegar o valor da posição topo de nossa barra de menu, para isso iremos utilizar o element.offsetTop (offsetTop irá retornar um valor em número)

Criaremos agora uma função com a seguinte condição if else mostrada na imagem:

window.pageYOffset retorna um valor número em pixels quando um usuário dá um scroll na página pra cima ou para baixo. Seu valor 0 indica que o documento está alinhado com o topo da janela da página.

Em nossa condição criada, caso for true iremos adicionar nossa classe sticky criada no CSS, caso a condição seja false iremos removê-la.

Agora iremos adicionar o window.scroll, assim a função irá ser executada a cada momento em que o usuário der um scroll e que o valor número em pixels de cada scroll seja maior ou igual ao valor em pixels do topo da nossa barra de menu.
