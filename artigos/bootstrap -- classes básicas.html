Bootstrap 5

Bootstrap é um framework CSS criado pelos funcionários do Twitter como forma de criar um padrão de estilo nos desenvolvimentos dos projetos dentro da empresa. Logo depois de um tempo de sua criação o Bootstrap foi lançado como um framework open source pelo Twitter.

Hoje quero mostrar aqui algumas classes bem básicas e também interessantes do Bootstrap 5. Irei mostrar imagens do código e também o resultado de cada código na tela do browser.

Heading

Usar o .h1 - .h6 no Bootstrap é uma boa prática caso você queira manter um padrão de estilização dos headings dentro da página.

<p class="h1"> Bootstrap Heading</p>

<p class="h2">Bootstrap Heading</p>

<p class="h3">Bootstrap Heading</p>

<p class="h4">Bootstrap Heading</p>

<p class="h5">Bootstrap Heading</p>

<p class="h6">Bootstrap Heading</p>

Display heading

O display é uma ótima opção para usar no Heading caso você queira que o título se destaque mais, pois ele aumenta o tamanho padrão do seu heading e o deixa levemente mais largo também.

<h1 class="display-1">Display 1</h1>

<h1 class="display-2">Display 2</h1>

<h1 class="display-3">Display 3</h1>

<h1 class="display-4">Display-4</h1>

Lead

O lead basicamente faz o seu parágrafo se destacar dos parágrafos normais.

<p class="lead">Vivamus sagittis lacus vel augue laoreet rutrum faucibus dolor auctor. Duis mollis, est non commodo luctus.</p>

Text-align

Podemos alinhar nosso texto no Bootstrap usando os seguintes códigos:

  • text-start (Equivale a text-left 'Padrão')
  • Text left
  • text-center
  • Text center
  • text-end (Equivale a text-right)
  • Text right

Inline

Remover os marcadores das listas e adicionar um leve margin , para isso irems usar duas classes .list-inline e .list-inline-item.

<li class="list-inline-item">Lorem ipsum</li>

<li class="list-inline-item">Phasellus iaculis</li>

<li class="list-inline-item">Nulla volutpat</li>
