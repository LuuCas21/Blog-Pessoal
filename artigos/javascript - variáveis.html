Variáveis em Javascript

Neste artigo irei dar uma breve explicação sobre variáveis em JavaScript. Pois é um dos temas fundamentais na linguagem de programação.

Por que utilizar variáveis ?

As variáveis são muito importantes na programação, nelas colocamos valores que podem ser reutilizados posteriormente ou também podemos declará-los para uso futuro, com isso, o nosso código fica mais fácil de se ler e bem compreendido.

Como definir variáveis no Javascript ?

A declaração de variáveis no Javascript é bem diferente de algumas outras linguagens como no Java, pois no JS não precisamos declarar o tipo de variável que estamos declarando, ou seja, o próprio JS faz a tipagem dinâmicamente. "Porém isso não significa que não exista tipos diferentes de variáveis no Javascript".

Exemplo de Declaração de variável:

const idioma = "português";

Javascript é Case Sensitive

Isso significa que Javascript é sensível a letras maíusculas e minúsculas, por assim dizer.

Exemplos para melhor compreensão:

const Idioma = "português"; é totalmente diferente de const idioma = "português";

Tipos de variáveis em JavaScript

  • String - É basicamente um texto puro.
  • Number - É basicamente números
  • Boolean - São operadores True e False.
  • Array - É uma estrutura de dados que serve para armazenar uma coleção de valores de qualquer tipo.
  • Object - Conjunto de atributos aninhados a uma variável denomina-se um objeto.
  • Function - Em Javascript é possível declarar uma variável como uma função, podendo fazer operações e retornando o valor para a variável de declaração.
